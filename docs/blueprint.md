# **App Name**: AlgoTradeAI

## Core Features:

- Live Trading Dashboard: Real-time dashboard displaying account balance, open positions, trading history, and AI-driven trading signals, using TradingView charting tools and the WalletConnect SDK.
- WalletConnect Integration: Prioritize Solana for wallet connections, while also enabling users to connect other cryptocurrency wallets (MetaMask, Trust Wallet, etc.) to deposit funds for trading and securely sign transactions.
- AI-Powered Strategy Generator: Allow users to create custom trading strategies by describing their desired rules; the AI tool translates these descriptions into executable trading logic, leveraging historical market data for strategy refinement.
- Profit Storage on BSC: Profits generated from trading will be stored on the Binance Smart Chain (BSC) for efficient and cost-effective transactions.
- Automatic Blockchain Selection for Transfers: The application will automatically select the appropriate blockchain for transferring funds based on the user's wallet and the transaction type.
- Performance Monitoring: Visualize performance metrics, including success rate, gains, and losses, and error logs that aid in auto-learning for future decisions.
- Preconfigured Strategies: Allows configuration of diverse technical indicators such as SMA, EMA, RSI, MACD and Bollinger Bands as well as allowing Breakout strategies.
- Sandbox Testing Environment: Enable users to test their trading strategies in a simulated environment with historical data from CoinDesk, TradingView and Gemini, providing risk-free evaluation and refinement of their approaches.

## Style Guidelines:

- Primary color: Deep, saturated purple (#6A459C) to convey sophistication and innovation.
- Background color: Dark, desaturated purple (#26202E) for an immersive experience.
- Accent color: Electric blue (#7DF9FF) to highlight interactive elements.
- Headline font: 'Space Grotesk' (sans-serif) for a tech-forward, modern look. Body text: 'Inter' (sans-serif) for a clean, readable interface.
- Code font: 'Source Code Pro' (monospace) for clear code presentation.
- Use minimalist icons with a line art style to maintain a clean and modern interface. Icons should be functional and clearly represent their actions.
- Subtle transitions and animations to provide feedback on interactions without being distracting. For example, a smooth animation when new data loads or when strategies are activated/deactivated.
